MicroCART (CPRE 491: Senior Design)

  • MicroCART is an ongoing senior design project "To create a modular platform for research in controls and embedded systems"
  • I am currently serving as the Test Station sub-team leader, and I'm working to create a platform for testing the drone's sensors and controls
  • The project uses C programming language for embedded code on the drone and test station, and my sub-team is using Fusion360 for 3D modeling

Simple Graphics Processor (CPRE 480: Computer Graphics Architecture)

  • The Simple Graphics Processor (SGP) was the semester-long group project to create a simple, OpenGL compliant GPU on an Artix7 FPGA
  • We used Vivado 2020.1 and VHDL to create the GPU, along with some C for the graphics driver code

Pipelined MIPS Processor (CPRE 381: Computer Org. & Assembly Level Programming)

  • Created a basic MIPS ISA processor with VHDL
  • VHDL was used to create the various components (ALU, control logic, register file), and ModelSim was used to simulate and test them.
